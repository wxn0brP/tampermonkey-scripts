# Tampermonkey Scripts Collection

My tampermonkey scripts

## Scripts

### GPT Un Overflow (`chatGptOverflow.js`)

**Platform:** ChatGPT
**Purpose:** Removes code block overflow issues on ChatGPT conversations by automatically clearing `max-height` restrictions.

### GitHub Remove Agents Nav Item (`githubAgents.js`)

**Platform:** GitHub  
**Purpose:** Removes the "Agents" navigation item from GitHub repository pages.

### Qwen.ai Copy Fix (`QwenCopyFix.js`)

**Platform:** Qwen.ai  
**Purpose:** Fixes Ctrl+C copying behavior on qwen.ai that incorrectly copies JSON instead of selected text.

### AI Chat Timeline Navigator (`timeline.js`)

**Platform:** ChatGPT, Qwen AI  
**Purpose:** Adds a visual timeline navigator on the right side of AI chat interfaces for quick navigation between messages.
